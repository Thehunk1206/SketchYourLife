About WebApp:

SketchYourlife is a simple Web Application which let's you to sketch your favourite image with just one click.
It uses a complex Image processing algorithm to turn your favourite image into a synthetic sketch.
More sketch like filter might be added in later release.
Please do contact us for any query.
